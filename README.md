# 🚗 Car Price Prediction

## 📌 Overview
This project aims to predict the selling price of cars based on various features such as brand, model, year, mileage, fuel type, transmission, and more. By leveraging machine learning techniques, the model provides accurate price estimations to assist buyers, sellers, and dealerships.

## 🛠️ Features
- Data preprocessing and cleaning
- Exploratory Data Analysis (EDA) with visualizations
- Feature engineering for improved model performance
- Multiple machine learning models tested (Linear Regression, Random Forest, XGBoost, etc.)
- Model evaluation using metrics like MAE, RMSE, and R²
- Deployment-ready pipeline for predictions
